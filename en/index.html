Toggle for Mode Dark .js

1. Include the CSS code at the beginning of your css file:

Obs: To use the css below, you must build your project with CSS variables.

:root {
/*Place the color variables of the Light theme here*/
--background : #eee ;
--text : #222 ;
}

[ data-theme = dark ]{
/*Place the color variables of the Dark theme here*/
--background : #161625 ;
--text : #eee ;
}

2. Add HTML Markup:

< div  id = "toggleButton" > < / div >

3. Add JavaScript before the tag /body:

< script src = " togglemodedark.js " > < / script >

4. Start ToggleButton:

< script > toggleModeDark (); < / script >

ToggleButton options:

Options
Type
Default
Description
id:
Text
"#toggleButton"
Change the ToggleButton id
toggleSize:
Number
30
Change the size of the ToggleButton
dotColor:
Text
"#fff"
Change the dot color of ToggleButton
activatedBackground:
Text
"#192351"
Changes the background color when activated
disabledBackground:
Text
"#f6ca69"
Changes the background color when activated disabled
textLightDark:
Boolean
false
Adds Light and Dark text to ToggleButton

Exemplos:

//Changing the Size of ToggleButton
toggleModeDark ({
toggleSize: 40
});

//Changing the size and background colors of ToggleButton
toggleModeDark ({
toggleSize: 40 ,
activatedBackground: "#eee" ,
disabledBackground: "#333" ,
});
